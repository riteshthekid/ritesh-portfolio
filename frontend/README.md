# Portfolio Frontend

React 19 + Vite + TypeScript frontend for Ritesh Patel's portfolio.

## Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:3000`. API requests are proxied to `http://localhost:4000` in dev.

## Build

```bash
npm run build
npm run preview
```

## Stack

- **React 19** with TypeScript
- **Vite** for bundling
- **Tailwind CSS v4** for styling
- **Motion** (Framer Motion) for animations
- **React Router v7** for routing
- **Lucide React** for icons
